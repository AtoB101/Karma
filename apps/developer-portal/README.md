# Developer portal — local deploy & Trust Layer

Static entry for **Deploy Locally**. Describes how to clone the public repository, run optional Docker, and (when published) use the `karma-agent` CLI.

**Canonical clone URL (this repository):**

```bash
git clone https://github.com/AtoB101/Karma.git
cd Karma
```

Commercial distributions may use alternate packaging names; always verify the Git remote you trust.
