KARMA
Developers

Proof receipts for paid agent actions.

Karma attaches verifiable receipts and evidence bundles to x402, AP2, and MCP transactions. Prove what was authorized, executed, and delivered — without replacing your payment stack.

Open Console GitHub

Wallet connection, evidence, and proof flows are inside Console. See the GitHub README for a 10-minute getting-started guide.

Three Primitives

Execution Receipt

Signed, hashable record of one agent/tool/API call: agent id, tool name, input hash, output hash, timestamp, status, optional payment reference.

Evidence Bundle

Portable audit package: collects receipts, payment references, delivery metadata, and verification hints into one verifiable unit.

Verification

Structural and rule-based checks on receipts and bundles. Confirm signatures, chain integrity, hash correctness, and completeness.
