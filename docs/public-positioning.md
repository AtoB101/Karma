# Karma Public Positioning

Karma is a non-custodial protection layer for agent-service payments.

## Positioning statement

Karma does not replace payment protocols.
Karma protects agent service payments after checkout.

## Public product focus

The first public product surface is:

- `Karma Guard for Agent Services`

It demonstrates:

- Buyer Protection
- Seller Bond
- Evidence Settlement
- Non-custodial Flow
- Karma Protected Badge

## What public docs include

- Integration entrypoints
- Public evidence bundle fields
- Public status machine
- Demo workflow

## What public docs do not include

- Private scoring weights
- Anti-fraud thresholds
- Arbitration internal weighting
- Private risk model internals
