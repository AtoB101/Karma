This repository uses a dual-licensing model to keep the project open for the
community while protecting commercial use.

Unless you have a separate commercial agreement, source code in this repository
is available under:

- GNU Affero General Public License v3.0 (`LICENSE`)

Scope:

- The root `LICENSE` applies repository-wide by default.
- Legacy per-folder license files are intentionally removed to avoid ambiguity.

Key points:

- You can run, study, modify, and redistribute under AGPLv3 terms.
- If you modify and provide the software as a network service, AGPLv3 requires
  offering corresponding source code to users of that service.

## 2) Commercial license

If your use case requires closed-source distribution/integration or terms that
are incompatible with AGPLv3 obligations, you must obtain a commercial license
from the project owner.

- Contact: `licensing@karmapay.cloud`
- Include:
  - Company / project name
  - Intended usage model (SaaS/on-prem/embedded)
  - Whether source code will remain private
  - Expected deployment scale

licenses. In case of conflict, third-party license terms apply to that
