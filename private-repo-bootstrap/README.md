# Karma Private Repository Bootstrap

This directory is generated by `scripts/private-repo-sync.sh` and is intended to be copied into a **private Git repository**.

## Suggested destination repository
- Name: `karma-internal` (or your preferred private repo)

## Directory structure
- `docs/private/` for internal investor, tokenomics, and audit appendix materials
- `scripts/private/` for private operations and partner integration scripts
- `examples/private/` for private demos and reproducible internal scenarios
- `outreach/` for customer/seller leads and business pipeline records

## Important
- Never push this content to the public `Karma` repository.
- Add branch protection + CODEOWNERS in private repo.
- Store runtime credentials in secret manager (not in git).
